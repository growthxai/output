Every release that introduces a breaking change adds a section to this page. Each section covers one version boundary (e.g. `v0.1` → `v0.2`). If you're jumping several versions, apply the sections in order.

## How to upgrade

The fastest path is the CLI:

```bash
npx output migrate
```

It reads the `@outputai/*` version in your `package.json`, finds the right section below, applies the changes, and runs your type checker.
